# Soniva

**Soniva** is a modern, low-cost mobile web app that helps people with hearing loss reconnect with the world. Designed for clarity, accessibility, and simplicity, Soniva provides real-time speech-to-text transcription, auditory training, tinnitus relief, and hearing health education â€” all from your smartphone.
